## 📖 Overview

This repository contains a comprehensive, self-contained **HTML question bank** for **EHS 701: Principles of Environmental Health**, a core course in the Master of Public Health (MPH) programme.

The question bank includes:
- **500 multiple‑choice questions** covering every topic from the lecture materials
- A complete **answer key** for self‑assessment
- **Detailed explanations** for every question to reinforce learning

All content is derived exclusively from the official EHS 701 lecture slides and is designed to test **application, analysis, evaluation, and synthesis** at a Master's level.
